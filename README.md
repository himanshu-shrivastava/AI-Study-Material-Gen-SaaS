## Getting Started :

### Installation :

Project :
```bash
git clone <REPOSITORY>
cd <PROJECT_FOLDER>
npm i
cp env.example .env.local  (update .env.local variables)
npm run dev
```
Run Drizzle :
```bash
npm run db:push
npm run db:studio
```
Run Inngest :
```bash
npx inngest-cli@latest dev
```
